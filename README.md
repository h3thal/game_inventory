# Игра-инвентарь

ТЗ: [TZ.md](TZ.md)

## Запуск проекта

### pnpm

```bash
pnpm install
pnpm dev
```

### npm

```bash
npm install
npm run dev
```

## О проекте

Проект — мини-инвентарь с игровой логикой. Цель — реализовать UI и поведение по требованиям из ТЗ.

## Стек

- Vue 3 (Composition API).
- TypeScript.
- Vite.
- Pinia.
- Tailwind CSS.
